# Установочная встреча пилота ОРБИТА
## Для PM пилотных продуктов | [дата]

---

## Слайд 1: Зачем мы здесь (3 мин)

**Проблема сегодня:**
- Каждый продукт работает по-своему
- Руководство не видит статус без запроса PM
- Приоритеты по голосу, не по данным
- Нет единого языка между командами

**Что меняем:**
ОРБИТА — единая система для всех продуктов портфеля.
Не заменяет YouTrack/Shiva. Добавляет портфельный уровень.

---

## Слайд 2: Как работает ОРБИТА (5 мин)

**7 Орбит (стадий зрелости):**
О·0 Заявка → О·1 Разведка → О·2 Прототип → О·3 Разгон → О·4 Экосистема → О·5 Плато → О·6 Архив

**Виток внутри каждой Орбиты:**
Исследование → Разработка → Оценка и ретро

**Gate Review:**
Комитет решает Go / No-Go по чеклисту обязательных критериев.

**Иерархия:**
Продукт → Цель (OKR) → Инициатива (RICE) → работа в YouTrack

## Слайд 6: Практика (30 мин)

**Задание для каждого PM:**
1. Открыть реестр → найти свой продукт (2 мин)
2. Проверить: правильная Орбита? Цели актуальны? (5 мин)
3. Открыть Gate-чеклист → оценить готовность к Gate (5 мин)
4. Создать 1 новую инициативу через «+ Создать» (5 мин)
5. Открыть «Как мы работаем» → найти свою стадию (3 мин)

**Если что-то непонятно или неудобно — говорите сразу. Это пилот, мы дорабатываем.**

## Слайд 7: Договорённости (10 мин)

**Еженедельный синк:**
- День: [определить на встрече]

**Обратная связь:**
- В любой момент: pmo@rwb.ru
- На синке: 5 мин на фидбек каждую неделю
