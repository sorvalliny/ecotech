ИИ как ускоритель, не замена процесса

ИИ автоматизирует рутину, освобождая PM/PMO для стратегических решений. Фреймворк остается источником истины.

Дорожная карта внедрения

Три фазы: от пилота до полной интеграции ИИ в процессы фреймворка

Фаза 1
Пилот (Q2 2026)
  • LLM-ассистент для PM: брифы, CustDev, конкурентный анализ
  • RAG по базе знаний ОРБИТА (хэндбук, фреймворк, шаблоны)
  • Автогенерация повестки комитета для PMO
  • Метрики: время на документацию, NPS пользователей ИИ
  • Участники: 2-3 PM из пилотных продуктов + 1 PMO
Фаза 2
Масштабирование (Q3 2026)
  • ИИ-подсказки в ОРБИТА UI: рекомендации по RICE, проверка артефактов
  • Автоотчеты по портфелю из данных реестра
  • Pre-review для руководителей: автопроверка полноты перед Gate
  • Метрики: adoption rate, экономия часов, качество артефактов
Фаза 3
Оптимизация (Q4 2026+)
  • Полуавтоматическое ведение карточек продуктов (ИИ обновляет, PMO подтверждает)
  • Прогнозирование рисков по паттернам портфеля
  • Рекомендательная система: «какие инициативы запустить» на основе целей
  • Метрики: % автоматизированных процессов, FTE-эквивалент сэкономленный
