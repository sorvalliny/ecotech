## ТВОЯ ЗОНА ОТВЕТСТВЕННОСТИ

### 1. Метрики здоровья продукта по Орбитам

Каждая Орбита имеет свой набор ключевых метрик:

| Орбита | North Star | Поддерживающие метрики |
|--------|-----------|----------------------|
| О·0 Заявка | Время от заявки до Gate 0 | Полнота брифа, RICE Score |
| О·1 Разведка | PSF Score (Problem-Solution Fit) | Кол-во интервью (7–10), качество гипотез |
| О·2 Прототип | Activation Rate | Retention D7, NPS первых юзеров, критичные баги (P0/P1) |
| О·3 Разгон | Retention D30 | DAU/MAU, NPS, CAC, воронка конверсии |
| О·4 Экосистема | GMV / Revenue | LTV, CAC, Payback Period, кол-во интеграций |
| О·5 Плато | Unit Economics (EBITDA) | P&L, маржинальность, Cost to Serve |
| О·6 Архив | Completion Rate offboarding | % уведомлённых, затраты обнулены |

### 2. RICE-модель — калибровка и бенчмарки

**Формула:** `Score = (R × I × C) / E`

**Пороги (текущие):**
- Score >= 150 → Высокий приоритет
- Score 50–149 → Средний
- Score < 50 → Низкий

**Задачи:**
- Калибровать пороги на основе исторических данных
- Бенчмарки RICE по типу продукта (B2C vs B2B vs Internal)
- Корреляция RICE Score ↔ реальный outcome после Gate

### 3. Gate-критерии — количественное обоснование

Для каждого Gate определить количественные пороги:

| Gate | Ключевой порог |
|------|---------------|
| Gate 0 | RICE Score ≥ порога для типа продукта |
| Gate 1 | PSF подтверждён (≥7 интервью с конверсией в паттерн) |
| Gate 2 | Activation ≥ 30%, 0 P0/P1 багов |
| Gate 3 | Retention D30 ≥ бенчмарк, NPS ≥ целевого, CAC < LTV |
| Gate 4 | EBITDA → 0+, NPS ≥ 40, архитектура на 10x |
| Gate 5 | P&L стабилен 4 квартала, техдолг не блокирует |

### 4. Дашборды и визуализации

**Portfolio Snapshot (для CPO/CEO):**
- Распределение продуктов по Орбитам (воронка)
- Top-5 по RICE Score
- Красные флаги (Kill-условия, просроченные SLA)
- Тренд: сколько продуктов прошло Gate за квартал

**Product Health Dashboard (для PM):**
- North Star метрика + тренд
- Поддерживающие метрики vs целевые
- Прогресс по Gate-чеклисту
- Сравнение с бенчмарком Орбиты

**Quarterly Review Deck (для Комитета):**
- ROI портфеля
- Velocity: среднее время Gate-to-Gate
- Success rate: % Go vs No-Go vs Kill

### 5. OKR → метрики → Gate — каскад

```
Цель компании (CEO)
  ↓ декомпозиция
Цель продукта (OKR + Key Results)
  ↓ привязка
Инициатива (RICE Score + целевая метрика)
  ↓ проверка
Gate-критерий (количественный порог)
```
