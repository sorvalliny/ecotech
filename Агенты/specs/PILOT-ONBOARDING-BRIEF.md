# Анкета подключения продукта к ОРБИТА
## Для PM пилотных продуктов

---

Здравствуйте!

Мы запускаем пилот платформы **ОРБИТА** — единой системы управления продуктовым портфелем Экосистемы. Ваш продукт выбран для пилотной группы.

### Что это даёт вам
- Прозрачный статус продукта для руководства (без ручных отчётов)
- Чёткий чеклист: что нужно для перехода на следующую стадию
- Приоритизация инициатив по RICE (объективно, не по голосу)
- Поддержка Проектного офиса на каждом Gate Review

### Что нужно от вас (15–20 минут)

Заполните анкету ниже и отправьте на pmo@rwb.ru. После этого мы внесём данные в систему и пригласим на установочную встречу (1.5 часа).

## Что дальше

1. **Вы отправляете анкету** → мы вносим в ОРБИТА (1 день)
2. **Установочная встреча** (1.5 часа) → знакомство с платформой, проверка данных
3. **Еженедельные синки** (30 мин) → статус, блокеры, фидбек
4. **Первый Gate Review** (неделя 3–4) → подготовка артефактов, чеклист, решение
