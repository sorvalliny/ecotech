
## 6. ДИЗАЙН-СИСТЕМА: ВИЗУАЛЬНЫЕ АКЦЕНТЫ

5 CSS-компонентов для ключевых тем платформы — разместить в соответствующих разделах:

```
.visual-initiative   → Инициатива   💡  фиолетовый градиент
.visual-goal         → Цель          🎯  зелёный акцент
.visual-planning     → Планирование  📅  синий
.visual-lifecycle    → Жизненный цикл    горизонтальная дорожка Gate 0→5
.visual-cycle        → Виток         🔄  SVG-круг 4 сектора
```

**`.visual-lifecycle`** — горизонтальная полоса с 6 этапами (Gate 0→5):
- Каждый Gate = кружок с номером + подпись
- Активный подсвечивается `--violet` + пульсация
- Соединительные линии между кружками

**`.visual-cycle`** — SVG-круг 4 сектора:
- Discovery `--violet` / Delivery `--teal` / Launch `--green` / Retro `--amber`
- При hover сектор увеличивается + появляется подпись
- Медленное вращение `animation: spin 20s linear infinite`

Размещение по файлам:
- `framework.html` → все 5 компонентов
- `framework-lite.html` → `.visual-cycle` + `.visual-lifecycle`
- `tracker.html` → `.visual-lifecycle` + `.visual-planning`
