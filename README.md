# ОРБИТА

Платформа запуска продуктов экосистемы RWB.

Фреймворк, шаблоны, трекер и поддержка Проектного офиса.

## Стек

- Vanilla HTML / CSS / JS — без сборщиков
- Деплой: GitHub Pages
- Шрифты: Orbitron (display), Exo 2 (body)
- Дизайн-система: `design-system/tokens.css`

## Структура

```
├── index.html              # Главная
├── business.html           # Для бизнеса
├── product.html            # Для продукта
├── terminology.html        # Глоссарий
├── nav.js                  # Универсальная навигация
├── design-system/          # Токены и компоненты
├── css/                    # Темы, адаптив, поиск
├── js/                     # Тема, поиск, хлебные крошки
├── Фреймворк/              # Продуктовый фреймворк
├── Продукту/               # Инструменты, стратегия, шаблоны
├── Проектный офис/         # Управляющий совет
└── data/                   # JSON-данные
```
