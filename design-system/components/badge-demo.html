Badge Component

Status badges for Gate progression and risk indicators. WCAG AA compliant (4.5:1+ contrast).

Gate Progression

Progressive darkening from neutral gray (Gate 0) through blue tones to green (Gate 5 — mature).

HTML
<span class="badge badge--gate-0">...Gate 0</span> <span class="badge badge--gate-3">...Gate 3</span> <span class="badge badge--gate-5">...Gate 5</span>
JavaScript
Badge.html('gate', 3) // → HTML string Badge.html('gate', 3, 'Запуск') // → custom label Badge.gate(5) // → DOM element
Risk / Status

Semantic status indicators for project health dashboards.

HTML
<span class="badge badge--on-track">...On Track</span> <span class="badge badge--at-risk">...At Risk</span> <span class="badge badge--paused">...Paused</span> <span class="badge badge--done">...Done</span>
JavaScript
Badge.html('risk', 'at-risk') // → HTML string Badge.html('risk', 'done', 'Готово') // → custom label Badge.risk('on-track') // → DOM element
